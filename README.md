Игра ASH-Legacy представляет собой открытый мир, где у человека есть индикатор здоровья, индикатор стамины, индикатор еды, возможность выбора оружия, возможность сесть в автомобиль и еще много чего. В игре есть режим радиации, которая не даст заскучать. Вот некоторые скриншоты из игры.
